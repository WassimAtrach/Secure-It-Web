# Secure-It 🛡️

Secure-It is a beginner-friendly cyber security awareness website designed to help everyday internet users understand common online threats and learn how to stay safe online. 

This repository contains the **home page** of the Secure-It project.

## 🌐 Live Demo
[https://WassimAtrach.github.io/Secure-It-Web/](https://wassimatrach.github.io/Secure-It-Web/)

## 🎯 Project Goals
* Raise awareness about common cyber threats.
* Teach non-technical users how to recognize scams.
* Encourage safe online habits through simple explanations.
* Provide an accessible introduction to cyber security concepts.

## 📄 Features
* **Hero section** with a clear call-to-action.
* **Mission overview**:
    * What Secure-It is.
    * What users will learn.
    * Who the platform is for.
* **Informational aside** with cyber safety tips.
* **Clean and semantic HTML5** structure.
* **Responsive layout** using Flexbox and media queries.

## 🛠️ Technologies Used
* **HTML5** (semantic structure)
* **CSS3**
* **Flexbox** for layout
* **Media queries** for responsiveness
* **JavaScript** (included for future interactivity)

## 📱 Responsive Design
The layout adapts to different screen sizes:
* **Desktop**: Main content and sidebar displayed side-by-side.
* **Mobile**: Content stacks vertically for better readability.

This behavior is implemented using **Flexbox** and a breakpoint at **768px**.
